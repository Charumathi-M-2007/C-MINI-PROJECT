# Mini Banking System

Mini Banking System is a console-based application developed using the C programming language and binary file handling techniques. The project is designed to perform basic banking operations such as account creation, deposit, withdrawal, balance enquiry, account deletion, and displaying account details.

The system uses random-access file handling to efficiently store and retrieve customer records from a binary file. It demonstrates important programming concepts such as structured programming, file handling, data validation, and random-access processing using functions like `fseek()`, `fread()`, and `fwrite()`.

## Features

* Create new bank accounts
* Deposit money
* Withdraw money
* Balance enquiry
* Delete account records
* Display all accounts
* Export account details to text file
* Random-access file handling using binary files

## Technologies Used

* C Programming
* Binary File Handling
* Random Access Files
* Structured Programming

## Files Used

* `bank.c` – Main source code
* `credit.dat` – Binary data file
* `accounts.txt` – Exported text file

## Concepts Implemented

* Structures
* Functions
* File Handling
* Random Access Processing
* Menu-Driven Programming
* Data Validation

## Conclusion

This project provides a basic implementation of a banking management system using C. It serves as a foundation for understanding file handling and random-access processing and can be further enhanced with advanced features such as transaction history, user authentication, and graphical interfaces.
